PTX VM in Browser

上传一个 .ptx，选中 kernel，然后直接运行

这个页面直接调用当前仓库里的 PTX VM。上传后会自动识别 entry kernel，支持标量参数和指针缓冲区参数， 并把执行后的缓冲区预览展示出来。

默认 Block 32 x 1 x 1
指针缓冲 支持 int/uint/float
固定地址观察 0x10000 / 0x20000

1. 选择 PTX

上传后会分析所有 entry kernel。

2. 配置并运行

支持选择 kernel、调整 grid / block，并为参数填值。

指针参数说明

指针参数会由后端自动分配设备内存。你可以选择缓冲区类型，设置元素个数，并填入逗号分隔的初始值。 如果初始值数量少于元素个数，剩余部分会自动补 0。

3. 执行结果

这里会展示标量参数、指针缓冲区预览，以及固定地址内存观察结果。
